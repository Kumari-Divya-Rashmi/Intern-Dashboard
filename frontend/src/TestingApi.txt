2. Register Intern API

Method:

POST

URL:

http://localhost:5000/api/auth/register

Body → raw → JSON:

{
  "name": "Kaushal Raj",
  "email": "kaushal@example.com",
  "password": "123456",
  "role": "intern"
}

Expected response:

{
  "success": true,
  "message": "User registered successfully",
  "token": "...",
  "user": {
    "id": "...",
    "name": "Kaushal Raj",
    "email": "kaushal@example.com",
    "role": "intern",
    "referralCode": "kaushalraj5577"
  }
}

Say:

This API creates a new intern. The password is hashed before saving to MongoDB.

If it says user already exists, change email:

{
  "name": "Kaushal Raj",
  "email": "kaushal2@example.com",
  "password": "123456",
  "role": "intern"
}
3. Login API

Method:

POST

URL:

http://localhost:5000/api/auth/login

Body:

{
  "email": "kaushal@example.com",
  "password": "123456"
}

Expected response:

{
  "success": true,
  "message": "Login successful",
  "token": "...",
  "user": {
    "id": "...",
    "name": "Kaushal Raj",
    "email": "kaushal@example.com",
    "role": "intern"
  }
}

Say:

This API checks the email and password. If credentials are valid, it returns a JWT token.

Copy the token.
